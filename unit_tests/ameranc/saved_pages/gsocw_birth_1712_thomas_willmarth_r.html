Description

This database includes all applications and supplemental applications to the General Society of Colonial Wars starting from their founding in 1893. Material is not included for individuals listed in the applications who were born after January 1, 1950. This database is searchable for given names and surnames, and includes records for provided births, baptisms, marriages, deaths and burials listed for persons in the line of eligibility for membership on the application forms.

Currently, the database includes applications from Member 1 through Member 2629. Additional application will be included as we process them. If an application between Members No. 1-2629 is excluded, it is either due to the privacy policy outlined above, or has been deemed as missing.

This database was created through a partnership of the General Society of Colonial Wars and American Ancestors.

Through the years, the General Society of Colonial Wars has established a large network of Colonial War descendants, with 32 state-specific chapters, across the United States. For more information about this society you can see the General Society of Colonial Wars web page.

The page numbers for the applications are in the format [general member number]_[supplement number]_[page number]. The numbers keep individual applications separate. For number 102, the first page is 102_00_1. Most applications have 4 pages, but some supplemental applications have up to 50 pages.

If you would like to request an application that is not listed in our database, please contact us at webmaster@nehgs.org with a general membership number for a copy of the application.

Search Tips

This database supports the following searchable fields:

  • First and last name
  • Year
  • Record type: Birth, Baptism, Intention, Marriage, Death and Burial
  • Location
  • Family member names: Spouses (where available)
  • Gen Member Number: The membership number in the General Society of Colonial Wars

Wild card characters are available for name fields. Helpful symbols to use in your name search include:

  • ? to match any single character
  • * to match any number of characters - Note that two characters must precede the * wildcard.
