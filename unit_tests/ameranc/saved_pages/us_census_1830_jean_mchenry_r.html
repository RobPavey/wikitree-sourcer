https://www.americanancestors.org/DB1562/rd/21640/28/434925549

Description

Name index to the population schedules listing the inhabitants of the United States in 1830. This was the fifth national census conducted since 1790. In the 1830 census, the head of each household is identified by name. The number of free white males and females are identified for each household based on age range. Households are organized based on the city, county, and state of location. This database contains more than 1.8 million records.

Keywords: MACensus, 10MillionNames

Search Tips

This index-only database supports the following searchable fields:

  • First and Last name
  • Location

All records are of type Census, and the date is the census year.

Wild card characters are available for name fields. Helpful symbols to use in your name search include:

  • ? to match any single character
  • * to match any number of characters - Note that two characters must precede the * wildcard.
