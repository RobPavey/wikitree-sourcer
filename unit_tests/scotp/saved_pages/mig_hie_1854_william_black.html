Search Options

If you do not know the name of the ship, please select 'all'.

If you do not know the port of arrival, please select 'all'.

If you do not know the departure port, please select 'all'.

If you do not know the county the emigrant came from, please select 'all'.

The spelling of the residence of the emigrant (before he or she emigrated) is as it is recorded in the passenger list. If you are unsure of the residence, please select 'all'. 

The name of the parish is not usually recorded in the passenger lists. In most cases the parish has been inferred from the estate and residence noted. If you are not sure of the parish, please select 'all'.
