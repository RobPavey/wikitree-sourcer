How to Add Sources

Include your sources below the biography like this:

== Sources ==
<references />
* Pavey, John. The Big Pavey Book. Chicago, IL: Pavey Publishing, 1914.

Include inline references like this:

The Pavey farm was 20 acres.
<ref>
The Big Pavey Book. Page 123.
</ref>

You could use the Citation Machine for help formatting citations.

[example] Formatting Tips

Link to anywhere on the web like this:

[https://www.familysearch.org/ FamilySearch]

Link to family members using their WikiTree IDs, e.g.:

[[Pavey-342|Harry Pavey]] [[Littlemore-13|Amelia Littlemore]]

Link to free-space profiles of family heirlooms, cemeteries, houses, etc., with links like this:

The Paveys had a [[Space:Pavey Family Farm|small farm]].

Experiment by copying-and-pasting any of the examples above into the text box and clicking "Preview".
